Name : Blockbuster database
URL : https://www.figure-eight.com/wp-content/uploads/2016/03/blockbuster-top_ten_movies_per_year_DFE.csv
Description : A data categorization job where we asked the crowd to find out information about the ten most popular movies, each year, for the past 40 years (1975-2015). 
Dataset includes:

Movie titles
Poster URLs for each
Genre information
Run time
MPAA ratings
IMDB rating
Rotten Tomato audience/critic rating
Box office receipts (adjusted for inflation)
